# M169 Abschlussprojekt

## Ziel
Lokale Infrastruktur als IaC mit Docker Compose:

- MediaWiki (firmeninternes Portal) auf Port 885
- Nextcloud (Open Source File Sharing)
- Gitea als internes Code Repository
- Portainer (Container Überwachung und Verwaltung)
- Persistenz für alle relevanten Daten (Volumes)
- Systemdokumentation und Konfiguration werden im privaten GitHub Repository versioniert

## Lokaler Pfad
~/gitrepo/m169/Abschlussprojekt

## Start (kommt später)
cd compose
docker compose up -d
